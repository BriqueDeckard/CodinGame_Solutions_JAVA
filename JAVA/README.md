## Command lines : 

### Compile : 
**Without packages**

-> javac MyFile.java 

**With packages**

-> javac -d . MyFile.java

### Execute : 

**Without packages**

-> java MyClass

**With packages**

-> java com.full.packages.path.MyClass

### Run a jar
**Without the classpath**
--> java -jar filename.jar
